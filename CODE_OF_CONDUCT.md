# NarrowCTI Code of Conduct

NarrowCTI is intended to be a professional, collaborative and welcoming open
source project for CTI, hunting, detection engineering and security platform
teams.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while still asking for evidence.
- Keep technical disagreement focused on the design, code or documentation.
- Help newcomers understand the project and contribution process.
- Protect sensitive information, including customer data, feed payloads,
  credentials and local lab artifacts.

## Unacceptable Behavior

- Harassment, threats, insults or discriminatory language.
- Publishing private information without permission.
- Sharing exploit details or credentials in public issues.
- Repeatedly derailing technical discussions.
- Submitting intentionally harmful code, payloads or configuration.

## Enforcement

Maintainers may moderate discussions, close issues, request edits, block
participation or take other reasonable action to protect the project and its
community.

If you see behavior that violates this code, contact the maintainers through a
private channel when possible. Security-sensitive reports should follow
`SECURITY.md`.
